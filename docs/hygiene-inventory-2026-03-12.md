## Artifact Classification

| Class | Paths / examples | Decision | Rationale |
|---|---|---|---|
| Runtime logs | `logs/*.log`, `host/rust/logs/*` | Ignore + periodic local cleanup | Operational data, not source assets |
| Rust build output | `**/target/`, `host/rust/target`, `desktop/apps/*/src-tauri/target` | Ignore + local cleanup | Deterministic build artifacts |
| Android build output | `android/**/build/`, `.gradle-user/` | Ignore + local cleanup | Generated by Gradle/AGP |
| Node frontend outputs | `desktop/apps/*/node_modules`, `desktop/apps/*/dist` | Ignore | Build/cache artifacts |
| Runtime scratch/state | `*.pid`, `*.tmp`, `*.cache`, `.wbeam_*` locals | Ignore | Environment/session specific |
| Keep markers | `logs/.gitkeep`, `host/rust/logs/.gitkeep` | Keep tracked | Preserve expected directory structure |
| Legacy docs/assets in archive | `archive/legacy/**` docs/assets | Keep for now (no removal in #31) | Requires product decision in Epic #3 |
| Suspicious helper candidate | `scripts/diagnostics/audodaignose` | Candidate for remove/fix in #33 | Naming + path quality suggests legacy/remnant helper |

## Cleanup Rules (Actionable)

1. Remove local generated artifacts whenever disk growth impacts local flow.
2. Never track runtime/build outputs in git.
3. Keep only structural placeholders (`.gitkeep`) in log directories.
4. Treat suspicious one-off helpers as removal candidates unless referenced by runtime/CI/docs.
5. Defer `archive/legacy` structural decisions to Epic #3 to avoid mixing scopes.
